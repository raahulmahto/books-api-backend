# books-api-backend

Production-ready Books API (Node.js, Express, MongoDB)

## Tech stack
- Node.js (>=18)
- Express
- MongoDB (Mongoose)
- Jest + Supertest (testing)
- Docker (local dev)

## Quick start (development)
1. Copy `.env.example` to `.env` and fill values.
2. `npm install`
3. `npm run dev` (uses nodemon)

## Scripts
- `npm run dev` - start in dev with nodemon
- `npm start` - start the server in production mode (as configured in package.json)
- `npm test` - run tests
- `npm run lint` - run linter
- `npm run build` - build if needed

## Contributing
- Follow ESLint/Prettier rules.
- Run tests & lint before pushing.
